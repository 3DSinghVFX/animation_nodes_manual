## Description

This node transform the input matrix by the the input transformation
matrix.

![image](transform_matrix_node.png)

## Inputs

  - **Matrix** - A matrix to transform.
  - **Transformation** - A transformation matrix.

## Outputs

  - **Matrix** - The transformed matrix.
