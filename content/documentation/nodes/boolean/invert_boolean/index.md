## Description

This node inverts the boolean. In other words, if the input is `False`
it becomes `True` and if it was `True` it will be `False`.

![image](invert_boolean_node.png)

## Inputs

- **Input** - The input boolean.

## Outputs

- **Output** - The inverted boolean.
