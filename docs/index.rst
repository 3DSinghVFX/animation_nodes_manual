Animation Nodes Documentation
=========================================================

Welcome to the official documentation of **Animation-Nodes**.

Animation nodes is a `Blender <http://www.blender.org/>`_ add-on to animate about everything in Blender by using nodes.

Using nodes grants you the ability to create complex and precise animations in a structured and centralized way.
