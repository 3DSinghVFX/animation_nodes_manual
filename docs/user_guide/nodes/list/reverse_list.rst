Reverse List
============

Description
-----------
This node flip the order of the elements of the list.The list ``[1,2,3]`` becomes ``[3,2,1]``.

.. image:: images/reverse_list_node.png
   :width: 160pt

Inputs
------

- **List** - An input list.

Outputs
-------

- **Reversed List** - The flipped list.
