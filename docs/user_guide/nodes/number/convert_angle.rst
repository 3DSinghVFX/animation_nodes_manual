Convert Angle
=============

Description
-----------
This node allows you to convert from *Degree* to *Radian* or from *Radian* to *Degree*.

.. image:: images/convert_angle_node.png
   :width: 160pt

Inputs
------

- **Degree/Radian** - The angle to convert.

Outputs
-------

- **Degree/Radian** - The converted angle.
