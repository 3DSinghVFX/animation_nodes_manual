Constant Falloff
================

Description
-----------

This node creates a falloff that associate a constant float for every object.

.. image:: images/constant_falloff_node.png

Inputs
------

- **Strength** - A float to associate to every object.

Outputs
-------

- **Falloff** - The actual falloff object.
