Create Edge Indices
===================

Description
-----------
This node create an edge indices which then can be used to create an edge indices list to create objects.

.. image:: images/create_edge_indices_node.png
   :width: 160pt

Inputs
------

- **Index 1** - The index of the first vertex.
- **Index 2** - The index of the second vertex.

Outputs
-------

- **Edge Indices** - The resulted edge indices.
