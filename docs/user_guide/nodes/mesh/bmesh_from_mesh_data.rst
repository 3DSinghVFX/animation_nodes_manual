Bmesh From Mesh Data
====================

Description
-----------
This node converts a mesh data block into a Bmesh block.

.. image:: images/bmesh_from_mesh_data_node.png
   :width: 160pt

Inputs
------

- **Mesh Data** - Mesh data to convert.

- **Bmesh** - The resulted Bmesh.
