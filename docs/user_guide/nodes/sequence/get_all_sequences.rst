Get All Sequences
=================

Description
-----------

This node returns all the sequence strips in the sequence editor.

.. image:: images/get_all_sequences_node.png
   :width: 160pt

Inputs
------

- **Scene** - A scene.

Outputs
-------

- **Sequences** - A list of all sequences in the sequence editor.
