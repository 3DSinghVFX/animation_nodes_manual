Text Block writer
=================

Description
-----------

This node writes a string to a blender text block.

.. image:: images/text_block_writer_node.png
   :width: 160pt

Inputs
------

- **Text Block** - A text block to write to.
- **Text** - A string to write.

Outputs
-------

- **Text Block** - The input text block.
