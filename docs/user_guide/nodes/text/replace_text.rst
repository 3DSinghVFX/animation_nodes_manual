Replace Text
============

Description
-----------

This node replaces a part of the input string by another string.

.. image:: images/replace_text_node.png
   :width: 160pt

Inputs
------

- **Text** - The input string.
- **Old** - The characters to replace.
- **New** - The new characters.

Outputs
-------

- **Text** - The output string.
