Copy Object Data
================

Description
-----------

This node copies the data of one object to another. (They become linked)

.. image:: images/copy_object_data_node.png
   :width: 160pt

Inputs
------

- **From** - An object to copy from.
- **To** - An object to copy to.

Outputs
-------

- **To** - The copied to object.
