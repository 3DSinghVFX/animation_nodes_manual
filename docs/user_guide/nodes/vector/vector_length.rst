Vector Length
=============

Description
-----------

This node calculates the magnitude of the input vector.

.. image:: images/vector_length_node.png
   :width: 160pt

Inputs
------

- **Vector** - A vector to calculate its magnitude.

Outputs
-------

- **Length** - The magnitude of the input vector.
