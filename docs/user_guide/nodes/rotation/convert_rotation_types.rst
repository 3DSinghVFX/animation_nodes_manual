Convert Rotation Types
======================

Description
-----------
this node convert from a rotation type to another. It support :

- Quaternion to Euler.
- Euler to Quaternion.
- Quaternion to Matrix.
- Matrix to Quaternion.
- Euler to Matrix.
- Matrix to Euler.
- Quaternion to Axis Angle.
- Axis Angle to Quaternion.
