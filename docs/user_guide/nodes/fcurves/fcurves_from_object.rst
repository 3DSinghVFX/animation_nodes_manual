FCurves From Object
===================

Description
-----------

This node returns a list of all FCurves in the input object.

.. image:: images/fcurves_from_object_node.png
   :width: 160pt

Inputs
------

- **Object** - An object to return its fcurves.

Outputs
-------

- **FCurves** - A list of all fcurves in the input object.
