Number To Boolean
=================

Description
-----------

This node return False if the input number is zero and True if otherwise.

.. image:: images/number_to_boolean_node.png
   :width: 160pt

Inputs
------

- **Number** - A number.

Outputs
-------

- **Boolean** - A boolean which is False if the input number is zero and True if otherwise.
