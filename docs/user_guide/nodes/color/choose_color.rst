Choose Color
============

Description
-----------
This node outputs color data chosen from the color picker.

.. image:: images/choose_color_node.png
   :width: 160pt

Inputs
------

- **Alpha** - The value of the alpha channel of the color.

Outputs
-------

- **Color** - The output color data.
